## Motivation

人类在复杂的3D环境中执行任务时，能够有效利用长期记忆。
然而，当前的LLM在动态、多房间的3D环境中进行有效规划和行动时面临挑战。
我们认为，这部分是由于LLM缺乏适当的3D空间-时间记忆建模能力。

## Contribution

*   **3DMem-Bench基准**: 引入了一个全面的3D具身任务基准，包含超过26,000条轨迹和2,892个具身任务、问答和描述任务，旨在评估智能体在3D环境中利用长期记忆进行推理的能力。
*   **3DLLM-Mem模型**: 提出了一种新颖的动态记忆管理和融合模型，用于LLM中的具身空间-时间推理和行动。
*   **记忆效率和任务相关性**: 该模型通过工作记忆token作为查询，选择性地关注并融合情景记忆中最有用的空间和时间特征，从而使智能体能够专注于任务相关信息，同时在复杂、长视距环境中保持记忆效率。

## Method

1.  **工作记忆 (Working Memory)**:
    *   表示智能体当前的观察结果。
    *   这些工作记忆token被用作查询 (queries)。
2.  **情景记忆 (Episodic Memory)**:
    *   存储智能体过去的观察和交互。
    *   包含丰富的空间和时间特征。
3.  **动态记忆管理与融合**:
    *   工作记忆token作为查询，用于选择性地关注 (selectively attend to) 情景记忆中的相关信息。
    *   通过融合机制，将情景记忆中最有用的空间和时间特征与当前观察结果结合。
    *   这种机制旨在平衡记忆效率和任务相关性，确保智能体在复杂环境中只关注必要信息。

## Comparison with Previous Work

*   **vs. 现有LLM**: 现有LLM在动态、多房间的3D环境中进行有效规划和行动时表现不佳，主要因为缺乏适当的3D空间-时间记忆建模。3DLLM-Mem通过引入动态记忆管理和融合机制，弥补了这一不足。
*   **vs. 现有基准**: 现有的具身AI基准可能未能充分评估智能体在复杂3D环境中利用长期记忆进行推理的能力。3DMem-Bench通过其大规模、多任务的设计，提供了更全面的评估。
*   **vs. 简单记忆策略 (如"Most Recent Memory"和"Retrieval-Augmented Memory")**: 3DLLM-Mem在记忆管理和融合上更为精细，能够选择性地关注任务相关信息，从而在各种任务中显著优于这些基线方法。

## Motivation

Humans excel at performing complex tasks by leveraging long-term memory across temporal and spatial experiences.
In contrast, current Large Language Models (LLMs) struggle to effectively plan and act in dynamic, multi-room 3D environments.
We posit that part of this limitation is due to the lack of proper 3D spatial-temporal memory modeling in LLMs.

## Contribution

*   **3DMem-Bench Benchmark**: We introduce a comprehensive benchmark comprising over 26,000 trajectories and 2,892 embodied tasks, question-answering, and captioning, designed to evaluate an agent's ability to reason over long-term memory in 3D environments.
*   **3DLLM-Mem Model**: We propose a novel dynamic memory management and fusion model for embodied spatial-temporal reasoning and actions in LLMs.
*   **Memory Efficiency and Task Relevance**: Our model uses working memory tokens, representing current observations, as queries to selectively attend to and fuse the most useful spatial and temporal features from episodic memory, which stores past observations and interactions. This allows the agent to focus on task-relevant information while maintaining memory efficiency in complex, long-horizon environments.

## Method

1.  **Working Memory**:
    *   Represents the agent's current observations.
    *   These working memory tokens are used as queries.
2.  **Episodic Memory**:
    *   Stores past observations and interactions of the agent.
    *   Contains rich spatial and temporal features.
3.  **Dynamic Memory Management and Fusion**:
    *   Working memory tokens act as queries to selectively attend to relevant information within the episodic memory.
    *   A fusion mechanism combines the most useful spatial and temporal features from episodic memory with current observations.
    *   This mechanism aims to balance memory efficiency and task relevance, ensuring the agent focuses only on necessary information in complex environments.

## Comparison with Previous Work

*   **vs. Existing LLMs**: Current LLMs struggle with effective planning and action in dynamic, multi-room 3D environments due to a lack of proper 3D spatial-temporal memory modeling. 3DLLM-Mem addresses this by introducing a dynamic memory management and fusion mechanism.
*   **vs. Existing Benchmarks**: Prior embodied AI benchmarks may not fully assess an agent's ability to reason over long-term memory in complex 3D environments. 3DMem-Bench, with its large-scale and multi-task design, provides a more comprehensive evaluation.
*   **vs. Simpler Memory Strategies (e.g., "Most Recent Memory" and "Retrieval-Augmented Memory")**: 3DLLM-Mem employs a more sophisticated memory management and fusion approach, enabling it to selectively focus on task-relevant information and significantly outperform these baseline methods across various tasks.
