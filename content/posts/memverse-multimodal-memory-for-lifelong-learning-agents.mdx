## Motivation

尽管大型语言和视觉模型取得了快速进展，但AI智能体仍面临一个根本性限制：它们无法可靠地记忆。

缺乏可靠的记忆会导致智能体灾难性地遗忘过去的经验，难以进行长程推理，并在多模态或交互式环境中难以保持连贯性。

当前的智能体在处理持续学习、适应性遗忘和有界记忆增长方面存在挑战，尤其是在需要实时响应的多模态场景中。

## Contribution

- **模型无关的即插即用记忆框架**: MemVerse提供了一个通用的记忆解决方案，可与任何现有的AI模型集成。
- **分层记忆系统**: 结合了快速的参数化召回和分层的检索式记忆，实现了可扩展和自适应的多模态智能。
- **结构化长期记忆**: 将原始多模态经验转化为以分层知识图谱组织的结构化长期记忆。
- **持续整合与自适应遗忘**: 设计支持记忆的持续更新、适应性遗忘和有界增长，以管理记忆容量。
- **周期性蒸馏机制**: 引入一种机制，将长期记忆中的关键知识压缩到参数模型中，实现快速、可微分的召回，同时保持可解释性。

## Method

MemVerse通过以下模块和机制实现其记忆功能：

1.  **短期记忆 (Short-Term Memory, STM)**:
    *   维护最近的上下文信息，用于即时交互和快速反应。
    *   类似于人类的短期工作记忆，处理当前正在进行的信息。

2.  **长期记忆 (Long-Term Memory, LTM)**:
    *   将原始多模态经验转化为结构化数据，组织成**分层知识图谱 (Hierarchical Knowledge Graphs)**。
    *   支持**持续整合 (Continual Consolidation)**：不断更新和扩充知识图谱。
    *   支持**自适应遗忘 (Adaptive Forgetting)**：根据重要性或使用频率，选择性地遗忘不重要的信息，以实现**有界记忆增长 (Bounded Memory Growth)**。

3.  **记忆检索与推理**:
    *   结合**参数化召回 (Parametric Recall)**：利用模型内部参数快速提取知识。
    *   结合**分层检索 (Hierarchical Retrieval)**：从结构化的知识图谱中高效地检索相关信息。

4.  **周期性蒸馏机制 (Periodic Distillation Mechanism)**:
    *   定期将长期记忆中的关键知识压缩并编码到参数模型中。
    *   这使得参数模型能够进行快速、可微分的知识召回，同时保持长期记忆的可解释性。
    *   这种机制有助于弥合快速参数化召回和丰富的检索式记忆之间的差距。

## Comparison with Previous Work

- **vs. 仅依赖参数化记忆的模型**: 传统模型容易遭受灾难性遗忘，难以处理长程推理和持续学习。MemVerse通过外部检索式记忆避免了这一问题。
- **vs. 仅依赖RAG (Retrieval-Augmented Generation) 的模型**: 纯RAG系统可能缺乏对知识的结构化理解和整合能力，且检索效率受限于索引结构。MemVerse通过分层知识图谱和蒸馏机制提供更深层次的记忆整合与快速召回。
- **vs. 缺乏多模态支持的记忆系统**: 许多现有记忆方案主要针对单一模态（如文本）。MemVerse明确设计为处理和整合多模态经验。
- **vs. 静态记忆系统**: 许多记忆系统是静态的，难以适应新信息或进行知识更新。MemVerse支持持续整合和自适应遗忘，实现动态记忆管理。

## Motivation

Despite rapid progress in large-scale language and vision models, AI agents still suffer from a fundamental limitation: they cannot reliably remember.

Without robust memory, agents catastrophically forget past experiences, struggle with long-horizon reasoning, and fail to operate coherently in multimodal or interactive environments.

Current agent designs face challenges in continual learning, adaptive forgetting, and bounded memory growth, especially when real-time demands in multimodal scenarios are present.

## Contribution

- **Model-agnostic, plug-and-play memory framework**: MemVerse offers a universal memory solution that can be integrated with any existing AI model.
- **Hierarchical memory system**: Bridges fast parametric recall with hierarchical retrieval-based memory, enabling scalable and adaptive multimodal intelligence.
- **Structured long-term memories**: Transforms raw multimodal experiences into structured long-term memories organized as hierarchical knowledge graphs.
- **Continual consolidation and adaptive forgetting**: The design supports continuous updating, adaptive forgetting, and bounded memory growth to manage memory capacity.
- **Periodic distillation mechanism**: Introduces a mechanism that compresses essential knowledge from long-term memory into the parametric model, allowing fast, differentiable recall while preserving interpretability.

## Method

MemVerse achieves its memory capabilities through the following modules and mechanisms:

1.  **Short-Term Memory (STM)**:
    *   Maintains recent context for immediate interactions and rapid responses.
    *   Analogous to human working memory, processing information currently in focus.

2.  **Long-Term Memory (LTM)**:
    *   Transforms raw multimodal experiences into structured data, organized as **Hierarchical Knowledge Graphs**.
    *   Supports **Continual Consolidation**: continuously updating and expanding the knowledge graphs.
    *   Enables **Adaptive Forgetting**: selectively forgetting less important information based on relevance or usage frequency to achieve **Bounded Memory Growth**.

3.  **Memory Retrieval and Reasoning**:
    *   Combines **Parametric Recall**: leveraging internal model parameters for fast knowledge extraction.
    *   Integrates **Hierarchical Retrieval**: efficiently retrieving relevant information from the structured knowledge graphs.

4.  **Periodic Distillation Mechanism**:
    *   Periodically compresses and encodes essential knowledge from the long-term memory into the parametric model.
    *   This allows the parametric model to perform fast, differentiable knowledge recall while maintaining the interpretability of the long-term memory.
    *   This mechanism helps bridge the gap between rapid parametric recall and rich, retrievable memory.

## Comparison with Previous Work

- **vs. Models relying solely on parametric memory**: Traditional models suffer from catastrophic forgetting and struggle with long-horizon reasoning and continual learning. MemVerse addresses this through external retrieval-based memory.
- **vs. Models relying solely on RAG (Retrieval-Augmented Generation)**: Pure RAG systems may lack structured understanding and integration capabilities for knowledge, and retrieval efficiency can be limited by index structure. MemVerse offers deeper memory integration and faster recall via hierarchical knowledge graphs and distillation.
- **vs. Memory systems lacking multimodal support**: Many existing memory solutions are primarily designed for a single modality (e.g., text). MemVerse is explicitly designed to process and integrate multimodal experiences.
- **vs. Static memory systems**: Many memory systems are static and struggle to adapt to new information or update knowledge. MemVerse supports continual consolidation and adaptive forgetting, enabling dynamic memory management.
