## Motivation

基础模型的无状态性限制了智能体系统持续学习的能力，而这对于长周期推理和适应性至关重要。
为了解决这一限制，智能体系统通常会整合记忆模块来保留和重用过去的经验，以期在测试时实现持续学习。
然而，大多数现有的记忆设计都是人工设计且固定的，这限制了它们适应真实世界任务多样性和非平稳性的能力。

## Contribution

- **自动化记忆设计**: 引入ALMA框架，通过元学习自动设计记忆模块，取代了传统的人工设计，从而减少了人力投入。
- **开放式搜索空间**: Meta Agent能够以开放式方式搜索表达为可执行代码的记忆设计，理论上可以发现任意记忆设计，包括数据库schema及其检索和更新机制。
- **持续学习能力**: 使得智能体系统能够成为跨越不同领域的持续学习者，有效应对真实世界任务的多样性和非平稳性。
- **超越SOTA性能**: 在四个序列决策领域的大量实验表明，学习到的记忆设计在所有基准测试中都比最先进的人工记忆设计更有效、更高效。

## Method

ALMA (Automated meta-Learning of Memory designs for Agentic systems) 框架通过以下步骤元学习记忆设计：

1.  **Meta Agent设计**: 引入一个Meta Agent，负责搜索和生成记忆设计。这些设计以可执行代码的形式表达，允许高度的灵活性和开放性。
2.  **记忆设计表达**: 记忆设计不仅包括数据库schema，还涵盖了其检索（retrieval）和更新（update）机制，确保了记忆模块的完整功能。
3.  **元学习过程**: Meta Agent通过在多个任务上进行学习和评估，迭代地优化记忆设计。目标是找到能够使智能体在多样化环境中实现持续学习的设计。
4.  **经验重用与适应**: 学习到的记忆设计使智能体能够有效地保留和重用过去的经验，从而在面对新任务或环境变化时，能够快速适应并持续学习。

## Motivation

The statelessness of foundation models bottlenecks agentic systems' ability to continually learn, a core capability for long-horizon reasoning and adaptation.
To address this limitation, agentic systems commonly incorporate memory modules to retain and reuse past experience, aiming for continual learning during test time.
However, most existing memory designs are human-crafted and fixed, which limits their ability to adapt to the diversity and non-stationarity of real-world tasks.

## Contribution

- **Automated Memory Design**: Introduces the ALMA framework, which meta-learns memory designs to replace hand-engineered memory designs, thereby minimizing human effort.
- **Open-Ended Search Space**: The Meta Agent searches over memory designs expressed as executable code in an open-ended manner, theoretically allowing the discovery of arbitrary memory designs, including database schemas as well as their retrieval and update mechanisms.
- **Continual Learning Capability**: Enables agentic systems to be continual learners across diverse domains, effectively addressing the diversity and non-stationarity of real-world tasks.
- **Outperforms SOTA Baselines**: Extensive experiments across four sequential decision-making domains demonstrate that the learned memory designs enable more effective and efficient learning from experience than state-of-the-art human-crafted memory designs on all benchmarks.

## Method

The ALMA (Automated meta-Learning of Memory designs for Agentic systems) framework meta-learns memory designs through the following steps:

1.  **Meta Agent Design**: A Meta Agent is introduced to search for and generate memory designs. These designs are expressed as executable code, allowing for high flexibility and open-endedness.
2.  **Memory Design Representation**: Memory designs encompass not only database schemas but also their retrieval and update mechanisms, ensuring a complete functional memory module.
3.  **Meta-Learning Process**: The Meta Agent iteratively optimizes memory designs by learning and evaluating them across multiple tasks. The goal is to find designs that enable continual learning in diverse environments.
4.  **Experience Reuse and Adaptation**: The learned memory designs empower agents to effectively retain and reuse past experiences, facilitating rapid adaptation and continual learning when facing new tasks or environmental changes.

## Comparison with Previous Work

- **vs. Human-Crafted Memory**: Existing methods largely rely on human experts to manually design and fix memory structures (e.g., episodic memory, RAG, KG). ALMA significantly reduces human intervention through an automated meta-learning process, discovering superior and more task-adaptive memory designs.
- **vs. Fixed Memory Mechanisms**: Traditional memory designs, once set, struggle to adapt to non-stationary environments. ALMA's meta-learning approach allows memory designs to dynamically adjust and optimize, addressing the diversity and changes in real-world tasks.
- **vs. Limited Generalization**: Many existing memory systems perform well on specific tasks but have limited cross-domain generalization. ALMA aims to enable agents to be continual learners across different domains by learning general memory design principles.
- **vs. Focus on Retrieval/Update Only**: Prior research might focus on a specific aspect of memory (e.g., RAG's retrieval mechanism). ALMA takes a holistic view of the entire memory lifecycle, including its structure (database schemas), retrieval, and update mechanisms.
