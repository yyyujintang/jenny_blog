# 欢迎来到我的博客

你好！欢迎来到我的个人博客。这里是我分享技术、生活和思考的地方。

## 关于这个博客

这个博客使用 **Next.js 14** 构建，具有以下特点：

- 🚀 **快速** - 基于 Next.js 的静态生成，加载速度极快
- 📱 **响应式** - 完美适配各种设备
- 🌙 **深色模式** - 自动适配系统主题
- ✍️ **Markdown 支持** - 使用 Markdown 写文章，简单方便
- 🎨 **现代化设计** - 使用 Tailwind CSS 打造美观界面

## 如何写文章

在 `content/posts/` 目录下创建 `.mdx` 文件即可。每篇文章需要包含 frontmatter：

\`\`\`yaml
---
title: 文章标题
date: 2024-01-15
excerpt: 文章摘要
tags:
  - 标签1
  - 标签2
---
\`\`\`

然后就可以开始写你的文章内容了！

## 部署

这个博客可以轻松部署到 **Vercel**：

1. 将代码推送到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署完成！

就是这么简单！
