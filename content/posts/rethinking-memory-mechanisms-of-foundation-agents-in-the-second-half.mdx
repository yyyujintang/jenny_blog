## Motivation

AI研究正从模型创新转向问题定义和真实世界评估。在“下半场”，核心挑战是智能体在长周期、动态、用户依赖环境中实现真正的实用性。
智能体面临上下文爆炸，需要持续积累、管理和选择性重用大量信息。记忆机制因此成为弥补实用性差距的关键解决方案。

## Method

本文通过对AI智能体记忆机制进行全面的综述，构建了一个统一的分析框架：

1.  **记忆基质 (Memory Substrate)**:
    *   **内部记忆 (Internal Memory)**: 指智能体内部固有的记忆能力，例如LLM的上下文窗口、KV cache等。
    *   **外部记忆 (External Memory)**: 指智能体可以访问的外部存储系统，如向量数据库、知识图谱 (KG) 等。

2.  **认知机制 (Cognitive Mechanism)**:
    *   **感觉记忆 (Sensory Memory)**: 短暂保留感官输入，用于平滑感知。
    *   **工作记忆 (Working Memory)**: 临时持有和处理当前信息，例如思维链 (CoT)。
    *   **情景记忆 (Episodic Memory)**: 记录特定经验的上下文，例如过去的交互日志。
    *   **语义记忆 (Semantic Memory)**: 存储关于世界的概念和事实知识，例如知识库。
    *   **程序记忆 (Procedural Memory)**: 存储技能和行动模式，例如可重用的工作流程或工具技能。

3.  **记忆主体 (Memory Subject)**:
    *   **以智能体为中心 (Agent-centric)**: 关注智能体自身的记忆需求，如规划、工具使用、状态跟踪等。
    *   **以用户为中心 (User-centric)**: 关注用户偏好、历史交互、个性化信息等。

本文还分析了记忆在不同智能体架构中的实现方式，并探讨了记忆操作（如写入、检索、更新、遗忘）的学习策略。最后，对现有评估方法进行了总结，并提出了未来的研究方向。

## Motivation

AI research is undergoing a paradigm shift, moving from prioritizing model innovations towards emphasizing problem definition and rigorous real-world evaluation. As the field enters the "second half," the central challenge becomes achieving real utility for agents in long-horizon, dynamic, and user-dependent environments.
Agents in these settings face context explosion and must continuously accumulate, manage, and selectively reuse large volumes of information across extended interactions. Memory, therefore, emerges as the critical solution to bridge this utility gap.

## Contribution

- **Unified View of Memory Mechanisms**: Presents a unified view of foundation agent memory along three dimensions: memory substrate (internal and external), cognitive mechanism (episodic, semantic, sensory, working, and procedural), and memory subject (agent-centric and user-centric).
- **Analysis of Memory Instantiation and Operations**: Analyzes how memory is instantiated and operated under different agent topologies, highlighting learning policies over memory operations.
- **Review of Evaluation Benchmarks and Metrics**: Provides a comprehensive review of evaluation benchmarks and metrics for assessing memory utility.
- **Open Challenges and Future Directions**: Outlines various open challenges and future research directions in the field of AI agent memory.

## Method

This paper constructs a unified analytical framework by providing a comprehensive survey of AI agent memory mechanisms:

1.  **Memory Substrate**:
    *   **Internal Memory**: Refers to the agent's inherent memory capabilities, such as the LLM's context window and KV cache.
    *   **External Memory**: Refers to external storage systems accessible by the agent, such as vector databases and knowledge graphs (KGs).

2.  **Cognitive Mechanism**:
    *   **Sensory Memory**: Brief retention of sensory inputs to smooth perception.
    *   **Working Memory**: Temporary holding and manipulation of current information, e.g., Chain-of-Thought (CoT).
    *   **Episodic Memory**: Contextual record of specific experiences, e.g., past interaction logs.
    *   **Semantic Memory**: Conceptual and factual knowledge about the world, e.g., a knowledge base.
    *   **Procedural Memory**: Skills and action patterns, e.g., reusable workflows or tool skills.

3.  **Memory Subject**:
    *   **Agent-centric**: Focuses on the agent's own memory needs, such as planning, tool use, and state tracking.
    *   **User-centric**: Focuses on user preferences, historical interactions, and personalized information.

The paper also analyzes how memory is implemented in different agent architectures and explores learning policies for memory operations (e.g., writing, retrieval, updating, forgetting). Finally, it summarizes existing evaluation methods and proposes future research directions.

## Comparison with Previous Work

As a survey paper, this work primarily organizes and synthesizes existing research by providing a unified framework, rather than directly comparing performance against specific models or algorithms.

-   **vs. Disparate Existing Research**: Current research on AI agent memory is often fragmented, focusing on specific aspects (e.g., RAG, long-context management, particular types of memory systems). This paper's contribution lies in integrating these disparate studies into a comprehensive three-dimensional framework, offering a more holistic and systematic perspective.
-   **vs. Traditional Memory Models**: Traditional AI or cognitive science memory models might focus on human cognition or specific tasks. This paper maps these concepts to foundation agent design, incorporating the characteristics and challenges of LLMs.
-   **vs. Simple RAG Approaches**: Many earlier works treated memory simply as a RAG mechanism. This paper goes beyond that scope, categorizing memory into various cognitive types and emphasizing the importance of memory management, learning policies, and user-centric memory.
