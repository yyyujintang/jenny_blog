## Motivation

构建一个通用型智能体是人工智能领域长期以来的愿景。现有智能体在许多领域取得了显著进展，但在开放世界中完成长周期任务时仍面临挑战。

我们认为这主要是因为智能体缺乏必要的“世界知识”和“多模态经验”，这些是指导智能体完成各种长周期任务的关键。

## Contribution

- **分层有向知识图谱 (Hierarchical Directed Knowledge Graph)**：将世界知识转化为结构化的知识图谱，使智能体能够显式地表示和学习世界知识。
- **抽象多模态经验池 (Abstracted Multimodal Experience Pool)**：总结历史信息，为智能体的in-context learning提供丰富的参考。
- **知识引导规划器 (Knowledge-guided Planner)**：结合上述混合多模态记忆模块，增强智能体在面对长周期任务时的规划能力。
- **经验驱动反思器 (Experience-Driven Reflector)**：结合上述混合多模态记忆模块，提升智能体在面对长周期任务时的反思能力。
- **Optimus-1智能体**: 基于混合多模态记忆模块构建的多模态智能体，在Minecraft等复杂环境中表现出色。

## Method

Optimus-1的核心在于其**混合多模态记忆模块 (Hybrid Multimodal Memory module)**，该模块包含两个主要组件：

### 1. 知识表示与学习

1.  **世界知识转化**: 将原始世界知识转化为**分层有向知识图谱 (Hierarchical Directed Knowledge Graph)**。
2.  **显式表示**: 知识图谱允许智能体以结构化的方式显式地表示和存储世界知识。
3.  **知识学习**: 智能体可以从这个图谱中学习和推理，以指导其行为。

### 2. 经验总结与参考

1.  **历史信息总结**: 将智能体的历史交互信息和经验总结为**抽象多模态经验池 (Abstracted Multimodal Experience Pool)**。
2.  **丰富参考**: 经验池为智能体提供丰富的多模态参考，用于in-context learning。
3.  **指导决策**: 智能体可以检索并利用这些经验来指导其在长周期任务中的规划和行动。

### 3. 智能体架构

1.  **Optimus-1**: 在混合多模态记忆模块之上，构建了一个完整的Optimus-1多模态智能体。
2.  **知识引导规划器 (Knowledge-guided Planner)**: 利用知识图谱进行更有效的任务规划。
3.  **经验驱动反思器 (Experience-Driven Reflector)**: 利用经验池进行任务反思和学习，以改进未来的行为。

## Comparison with Previous Work

- **vs. 现有智能体**: 现有智能体在开放世界长周期任务中表现不佳，主要原因在于缺乏必要的“世界知识”和“多模态经验”。Optimus-1通过引入混合多模态记忆模块，显式地解决了这些问题。

## Motivation

Building a general-purpose agent is a long-standing vision in artificial intelligence. While existing agents have made remarkable progress in many domains, they still struggle to complete long-horizon tasks in open-world environments.

We attribute this limitation to the lack of necessary world knowledge and multimodal experience, which are crucial for guiding agents through a variety of complex, long-horizon tasks.

## Contribution

- **Hierarchical Directed Knowledge Graph**: Transforms world knowledge into a structured graph, allowing agents to explicitly represent and learn world knowledge.
- **Abstracted Multimodal Experience Pool**: Summarizes historical information to provide agents with rich references for in-context learning.
- **Knowledge-guided Planner**: Integrates with the Hybrid Multimodal Memory module to enhance the agent's planning capabilities for long-horizon tasks.
- **Experience-Driven Reflector**: Integrates with the Hybrid Multimodal Memory module to improve the agent's reflection and learning in the face of long-horizon tasks.
- **Optimus-1 Agent**: A multimodal agent constructed upon the Hybrid Multimodal Memory module, demonstrating superior performance in complex environments like Minecraft.

## Method

The core of Optimus-1 lies in its **Hybrid Multimodal Memory module**, which comprises two main components:

### 1. Knowledge Representation and Learning

1.  **World Knowledge Transformation**: Converts raw world knowledge into a **Hierarchical Directed Knowledge Graph**.
2.  **Explicit Representation**: The knowledge graph allows the agent to explicitly represent and store world knowledge in a structured manner.
3.  **Knowledge Learning**: The agent can learn and reason from this graph to guide its behavior.

### 2. Experience Summarization and Reference

1.  **Historical Information Summarization**: Summarizes the agent's historical interactions and experiences into an **Abstracted Multimodal Experience Pool**.
2.  **Rich References**: The experience pool provides the agent with rich multimodal references for in-context learning.
3.  **Decision Guidance**: The agent can retrieve and utilize these experiences to guide its planning and actions in long-horizon tasks.

### 3. Agent Architecture

1.  **Optimus-1**: A complete Optimus-1 multimodal agent is constructed on top of the Hybrid Multimodal Memory module.
2.  **Knowledge-guided Planner**: Utilizes the knowledge graph for more effective task planning.
3.  **Experience-Driven Reflector**: Leverages the experience pool for task reflection and learning to improve future behavior.

## Comparison with Previous Work

- **vs. Existing Agents**: Current agents perform poorly on open-world long-horizon tasks due to a lack of essential "world knowledge" and "multimodal experience." Optimus-1 explicitly addresses these issues by introducing its Hybrid Multimodal Memory module.
- **vs. GPT-4V Baseline**: Optimus-1 demonstrates strong generalization capabilities with its memory module, even when using various MLLMs as backbones, outperforming the GPT-4V baseline on many tasks.
- **vs. Other Baselines (e.g., DEPS, Jarvis-1)**: Optimus-1 significantly outperforms all existing agents on challenging long-horizon task benchmarks like Minecraft, exhibiting near human-level performance on many tasks.
