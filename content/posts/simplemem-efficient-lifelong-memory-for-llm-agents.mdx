## Motivation

LLM智能体在复杂环境中进行长期交互时，需要有效的记忆系统来管理历史经验。

现有方法要么保留完整的交互历史，导致大量冗余，要么依赖迭代推理来过滤噪声，产生高昂的token成本。

为了解决这些挑战，我们提出了SimpleMem，一个基于语义无损压缩的高效记忆框架。

## Contribution

- **语义结构化压缩 (Semantic Structured Compression)**: 将非结构化交互提炼为紧凑、多视图索引的记忆单元，最大化信息密度。
- **在线语义合成 (Online Semantic Synthesis)**: 在会话内部即时整合相关上下文，形成统一的抽象表示，消除冗余。
- **意图感知检索规划 (Intent-Aware Retrieval Planning)**: 推断搜索意图，动态确定检索范围，高效构建精确上下文。

## Method

SimpleMem采用三阶段管道设计，旨在最大化信息密度和token利用率：

1.  **语义结构化压缩 (Semantic Structured Compression)**:
    *   将原始的、非结构化的交互历史（如对话、观察、行动）转化为结构化的、语义丰富的记忆单元。
    *   这些记忆单元通过多视图索引（例如，语义层、词法层、符号层）进行组织，以便后续高效检索。

2.  **在线语义合成 (Online Semantic Synthesis)**:
    *   在当前会话进行时，实时识别并整合与新信息相关的旧记忆单元。
    *   通过抽象和概括，将这些相关信息融合成更紧凑、无冗余的表示，减少记忆库的膨胀。

3.  **意图感知检索规划 (Intent-Aware Retrieval Planning)**:
    *   根据LLM智能体的当前任务或查询，推断其潜在的搜索意图。
    *   基于推断出的意图，动态地规划检索策略，确定需要搜索的记忆视图和范围。
    *   高效地从压缩和合成的记忆中提取最相关的上下文，供LLM智能体使用。

## Comparison with Previous Work

-   **vs. 传统上下文扩展 (Passive Context Extension)**: SimpleMem通过语义压缩和合成，避免了保留完整交互历史所带来的巨大冗余和高昂token成本。
-   **vs. 迭代推理过滤 (Iterative Reasoning for Filtering)**: 传统方法依赖LLM进行多次推理来过滤噪声，SimpleMem则通过预先的语义处理和意图感知检索，显著降低了推理时的token消耗。
-   **vs. 其他记忆框架 (e.g., MemGPT, A-Mem, LightMem)**: SimpleMem在保持高准确性的同时，实现了更优的检索效率和更低的推理成本，在性能和效率之间取得了更好的平衡。

## Motivation

To support long-term interaction in complex environments, LLM agents require robust memory systems that effectively manage historical experiences.

Existing approaches often fall into two categories: retaining full interaction histories via passive context extension, which leads to substantial redundancy, or relying on iterative reasoning to filter noise, incurring high token costs.

To address these critical challenges, we introduce SimpleMem, an efficient memory framework built upon semantic lossless compression.

## Contribution

-   **Semantic Structured Compression**: Distills unstructured interactions into compact, multi-view indexed memory units, maximizing information density.
-   **Online Semantic Synthesis**: An intra-session process that instantly integrates related context into unified abstract representations to eliminate redundancy.
-   **Intent-Aware Retrieval Planning**: Infers search intent to dynamically determine retrieval scope and construct precise context efficiently.

## Method

SimpleMem employs a three-stage pipeline designed to maximize information density and token utilization:

1.  **Semantic Structured Compression**:
    *   Transforms raw, unstructured interaction histories (e.g., dialogues, observations, actions) into structured, semantically rich memory units.
    *   These memory units are organized with multi-view indexing (e.g., semantic, lexical, symbolic layers) for efficient subsequent retrieval.

2.  **Online Semantic Synthesis**:
    *   During an ongoing session, this stage identifies and integrates related older memory units with new information in real-time.
    *   It synthesizes these related pieces into more compact, non-redundant abstract representations through summarization and generalization, preventing memory bloat.

3.  **Intent-Aware Retrieval Planning**:
    *   Based on the LLM agent's current task or query, it infers the underlying search intent.
    *   This inferred intent dynamically guides the retrieval strategy, determining which memory views and scopes are most relevant for the current context.
    *   It efficiently extracts the most precise and relevant context from the compressed and synthesized memory for the LLM agent to utilize.

## Comparison with Previous Work

-   **vs. Passive Context Extension**: SimpleMem avoids the significant redundancy and high token costs associated with retaining full interaction histories by employing semantic compression and synthesis.
-   **vs. Iterative Reasoning for Filtering**: While prior methods rely on multiple LLM reasoning steps to filter noise, SimpleMem significantly reduces inference-time token consumption through its pre-processing semantic structuring and intent-aware retrieval.
-   **vs. Other Memory Frameworks (e.g., MemGPT, A-Mem, LightMem)**: SimpleMem achieves a superior balance between performance and efficiency, delivering higher accuracy while maintaining better retrieval efficiency and lower inference costs.
