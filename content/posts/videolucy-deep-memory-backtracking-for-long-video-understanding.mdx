## Motivation

长视频理解中的Agent-based LLM系统面临两大挑战：一是它们通常在单个帧上进行建模和推理，难以捕捉连续帧的时间上下文。
二是为降低密集帧级标注成本，这些系统采用稀疏帧采样，这可能导致关键信息丢失。
本研究旨在通过模拟人类从粗到细的回忆过程，解决这些限制，从而实现对长视频的有效时间理解和细节保留。

## Contribution

*   **分层记忆结构 (Hierarchical Memory Structure)**：受人类回忆过程启发，VideoLucy采用具有渐进粒度的分层记忆结构，明确定义了不同层次记忆的细节水平和时间范围。
*   **Agent-based 迭代回溯机制 (Agent-based Iterative Backtracking Mechanism)**：通过Agent驱动的迭代回溯，系统性地挖掘视频中与问题相关的深层记忆，直到收集到足够信息以提供可靠答案。
*   **新基准 EgoMem (New Benchmark EgoMem)**：引入了EgoMem，一个用于长视频理解的新基准，旨在全面评估模型理解复杂事件和捕捉极长视频中细粒度细节的能力。
*   **卓越的性能 (Superior Performance)**：VideoLucy在多个长视频理解基准测试中显著优于SOTA方法，甚至超越了最新的专有模型如GPT-4o。

## Method

VideoLucy是一个深度记忆回溯框架，其核心在于模拟人类从粗到细的回忆过程，通过Agent-based迭代机制来理解长视频。

1.  **分层记忆初始化 (Hierarchical Memory Initialization)**：
    *   首先，系统对长视频进行稀疏采样，并利用LLM生成粗粒度的视频描述，形成初始的“粗记忆”。
    *   这些粗记忆在时间上覆盖整个视频，但细节较少，类似于人类对事件的初步印象。
2.  **Agent-based 迭代回溯 (Agent-based Iterative Backtracking)**：
    *   当接收到用户问题时，一个“定位Agent”会根据当前记忆（初始粗记忆或后续细化记忆）判断哪些时间段可能包含与问题相关的关键信息。
    *   如果现有信息不足以自信地回答问题，定位Agent会选择一个最相关的粗粒度时间段，并指示一个“指令Agent”对该时间段进行更密集的采样和更详细的描述。
    *   这个过程会生成更细粒度的记忆，并将其添加到记忆结构中。
    *   系统会迭代执行这个“定位-细化-整合”的循环，直到收集到足够的信息来提供一个自信的答案。
3.  **记忆整合与推理 (Memory Integration and Reasoning)**：
    *   在每次迭代中，所有收集到的记忆（从粗到细）都会被LLM整合，用于回答问题。
    *   这种机制确保了在保持时间上下文的同时，能够深入挖掘关键细节，避免了稀疏采样可能带来的信息丢失。

## Comparison with Previous Work

*   **vs. 传统帧级处理方法 (vs. Traditional Frame-level Processing Methods)**：
    *   传统方法通常独立处理视频帧，难以捕捉连续帧之间的时间依赖和上下文。VideoLucy通过其分层记忆结构和迭代回溯机制，显式地建模时间上下文，并能从粗到细地理解事件的演变。

*   **vs. 其他Agent-based系统 (vs. Other Agent-based Systems)**：
    *   VideoLucy的独特之处在于其受人类回忆启发的“从粗到细”的分层记忆管理和迭代回溯策略，这使得它在处理长视频的复杂性和细节捕捉方面优于许多现有Agent-based系统。

## Motivation

Agent-based LLM systems for long video understanding face two major challenges. Firstly, they typically perform modeling and reasoning on individual frames, struggling to capture the temporal context of consecutive frames.
Secondly, to reduce the cost of dense frame-level captioning, these systems adopt sparse frame sampling, which risks discarding crucial information.
This research aims to overcome these limitations by mimicking the human recollection process from coarse to fine, thereby enabling effective temporal understanding and critical detail preservation in long videos.

## Contribution

*   **Hierarchical Memory Structure**: Inspired by human recollection, VideoLucy employs a hierarchical memory structure with progressive granularity, explicitly defining the detail level and temporal scope of memory at different depths.
*   **Agent-based Iterative Backtracking Mechanism**: Through an agent-driven iterative backtracking mechanism, VideoLucy systematically mines video-wide, question-relevant deep memories until sufficient information is gathered to provide a confident answer.
*   **New Benchmark EgoMem**: We introduce EgoMem, a new benchmark for long video understanding, designed to comprehensively evaluate a model's ability to understand complex events and capture fine-grained details in extremely long videos.
*   **Superior Performance**: VideoLucy significantly outperforms state-of-the-art methods on multiple long video understanding benchmarks, achieving performance even surpassing the latest proprietary models such as GPT-4o.

## Method

VideoLucy is a deep memory backtracking framework whose core lies in simulating the human recollection process from coarse to fine, using an agent-based iterative mechanism to understand long videos.

1.  **Hierarchical Memory Initialization**:
    *   First, the system sparsely samples the long video and uses an LLM to generate coarse-grained video descriptions, forming initial "coarse memories."
    *   These coarse memories cover the entire video temporally but lack detail, similar to a human's initial impression of an event.
2.  **Agent-based Iterative Backtracking**:
    *   Upon receiving a user query, a "localization agent" assesses which time periods, based on current memories (initial coarse memories or subsequent refined memories), likely contain key information relevant to the question.
    *   If existing information is insufficient for a confident answer, the localization agent selects the most relevant coarse-grained time period and instructs an "instruction agent" to perform more dense sampling and detailed description for that specific period.
    *   This process generates finer-grained memories, which are then added to the memory structure.
    *   The system iteratively performs this "localize-refine-integrate" loop until enough information is gathered to provide a confident answer.
3.  **Memory Integration and Reasoning**:
    *   In each iteration, all collected memories (from coarse to fine) are integrated by the LLM to answer the question.
    *   This mechanism ensures that critical details can be deeply mined while maintaining temporal context, effectively mitigating information loss from sparse sampling.

## Comparison with Previous Work

*   **vs. Traditional Frame-level Processing Methods**:
    *   Traditional methods often process video frames independently, struggling to capture temporal dependencies and context between consecutive frames. VideoLucy explicitly models temporal context through its hierarchical memory structure and iterative backtracking, understanding event evolution from coarse to fine.
*   **vs. Sparse Sampling Methods**:
    *   Existing agent-based systems often use sparse frame sampling to reduce costs, but this can lead to the loss of crucial details. VideoLucy's agent-driven "backtracking" mechanism allows for fine-grained sampling of specific time periods when needed, effectively alleviating the problem of information loss.
*   **vs. Other Agent-based Systems**:
    *   VideoLucy's unique "coarse-to-fine" hierarchical memory management and iterative backtracking strategy, inspired by human recollection, allows it to outperform many existing agent-based systems in handling the complexity and detail capture of long videos.
