## Motivation

目前大多数现有基准主要关注基于文本的agent，忽略了许多需要视觉信息才能有效解决的自然任务。
鉴于大多数计算机界面都迎合人类感知，视觉信息通常以文本模型难以有效利用的方式增强文本数据。
为了弥合这一差距，我们引入了VisualWebArena，一个旨在评估多模态网页agent在真实“视觉接地任务”中表现的基准。

## Contribution

- **VisualWebArena基准**: 提出了一个全新的基准，用于评估多模态agent在需要视觉理解的复杂网页任务中的表现。
- **多样化任务集**: 包含一系列多样且复杂的网页任务，这些任务旨在评估自主多模态agent的各种能力。
- **全面的评估框架**: 提供了一个评估多模态自主语言agent的框架，并为构建更强大的网页自主agent提供了见解。
- **识别现有模型局限性**: 通过广泛的定量和定性分析，揭示了文本LLM agent的局限性以及最先进多模态语言agent的能力差距。
- **公开可用资源**: 代码、基线模型和数据均已公开，促进了该领域的研究和开发。

## Method

VisualWebArena基准旨在评估agent在真实视觉网页任务中的表现。
1. **任务设计**: 任务要求agent准确处理图像-文本输入，解释自然语言指令，并在网站上执行操作以完成用户定义的目标。
2. **环境设置**: agent在一个模拟的网页环境中运行，该环境能够呈现视觉信息并响应agent的动作。
3. **评估指标**: 主要通过任务成功率来衡量agent的表现，同时进行定性分析以深入了解其行为和局限性。
4. **Agent类型**: 评估了多种LLM-based自主agent，包括纯文本模型和多模态模型。

## Comparison with Previous Work

- **vs. 现有文本基准**: 大多数现有基准主要关注文本型agent，而VisualWebArena专注于需要视觉信息的真实网页任务，弥补了这一空白。
- **vs. 缺乏视觉信息的任务**: 现有基准未能有效评估agent利用视觉信息增强文本数据以解决复杂任务的能力，VisualWebArena通过其视觉接地任务解决了这个问题。
- **vs. 简单网页任务**: VisualWebArena的任务集更加多样和复杂，旨在评估更广泛的自主多模态agent能力，而非简单的导航或信息提取。

## Motivation

The majority of existing benchmarks primarily focus on text-based agents, neglecting many natural tasks that require visual information to effectively solve.
Given that most computer interfaces cater to human perception, visual information often augments textual data in ways that text-only models struggle to harness effectively.
To bridge this gap, we introduce VisualWebArena, a benchmark designed to assess the performance of multimodal web agents on realistic "visually grounded tasks".

## Contribution

- **VisualWebArena Benchmark**: Introduced a novel benchmark for evaluating multimodal agents on complex web tasks that require visual understanding.
- **Diverse Task Set**: Comprises a set of diverse and complex web-based tasks designed to evaluate various capabilities of autonomous multimodal agents.
- **Comprehensive Evaluation Framework**: Provides a framework for evaluating multimodal autonomous language agents and offers insights towards building stronger autonomous agents for the web.
- **Identification of Model Limitations**: Through extensive quantitative and qualitative analysis, revealed limitations of text-only LLM agents and gaps in the capabilities of state-of-the-art multimodal language agents.
- **Publicly Available Resources**: Code, baseline models, and data are publicly available to foster research and development in this domain.

## Method

The VisualWebArena benchmark is designed to evaluate agents on realistic visual web tasks.
1. **Task Design**: Tasks require agents to accurately process image-text inputs, interpret natural language instructions, and execute actions on websites to accomplish user-defined objectives.
2. **Environment Setup**: Agents operate within a simulated web environment that can render visual information and respond to agent actions.
3. **Evaluation Metrics**: Performance is primarily measured by task success rate, complemented by qualitative analysis to gain insights into agent behavior and limitations.
4. **Agent Types**: An extensive evaluation was conducted on various LLM-based autonomous agents, including both text-only and multimodal models.

## Comparison with Previous Work

- **vs. Existing Text-based Benchmarks**: Most existing benchmarks primarily focus on text-based agents, whereas VisualWebArena addresses the need for visually grounded realistic web tasks.
- **vs. Lack of Visual Information**: Prior benchmarks fail to effectively evaluate agents' ability to leverage visual information to augment textual data for complex tasks, a gap VisualWebArena fills with its visually grounded tasks.
- **vs. Simple Web Tasks**: VisualWebArena's task set is more diverse and complex, aiming to evaluate a broader range of autonomous multimodal agent capabilities beyond simple navigation or information extraction.
