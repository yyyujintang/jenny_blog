## Motivation

大型语言模型（LLM）的个性化旨在使模型输出与用户的独特偏好和观点保持一致。尽管现有方法已取得进展，但仍缺乏一个统一的理论框架来系统地理解有效个性化的驱动因素。

我们认为，认知科学中的双记忆模型（episodic memory和semantic memory）可以为LLM个性化提供一个强大的理论基础。通过将用户历史互动映射到episodic memory，将长期演变的用户信念映射到semantic memory，我们可以更全面地捕捉用户画像。

## Contribution

*   **统一的认知双记忆框架**: 首次将认知双记忆模型（episodic memory和semantic memory）系统地整合到LLM个性化中，并提出了PRIME框架。
*   **个性化思维能力**: 引入了受“慢思考”策略启发的个性化思维能力，进一步增强了模型的推理和响应质量。
*   **新基准数据集**: 构建了一个基于Reddit Change My View (CMV) 的长上下文个性化数据集，以解决现有基准的不足。
*   **全面的实验验证**: 在长上下文和短上下文场景中对PRIME进行了广泛验证，证明了其在捕捉动态个性化方面的有效性。

## Method

PRIME框架通过以下核心组件实现LLM个性化：

### 1. 记忆实例化 (Memory Instantiations)

*   **Episodic Memory (情景记忆)**: 镜像用户的历史互动，捕捉具体事件和上下文。这包括用户过去的对话、偏好选择等。
*   **Semantic Memory (语义记忆)**: 对应用户长期、演变的信念和偏好。通过对情景记忆的抽象和泛化形成，代表了用户更稳定的个性特征。

### 2. 个性化思维过程 (Personalized Thought Process)

*   **慢思考策略 (Slow Thinking Strategy)**: 借鉴人类的慢思考模式，PRIME在生成最终响应之前，会进行一个内部的、多步骤的推理过程。这个过程利用双记忆中的信息，更深入地理解用户意图和偏好，从而生成更细致、更个性化的输出。

## Comparison with Previous Work

*   **vs. 现有个性化方法**: 现有方法通常侧重于单一的记忆形式（如RAG或微调），缺乏一个统一的理论框架。PRIME通过整合认知双记忆，提供了更全面的用户画像。
*   **vs. RAG (Retrieval-Augmented Generation)**: RAG主要依赖于检索相关文档，类似于一种简单的episodic memory。PRIME在此基础上增加了semantic memory和个性化思维过程，能够捕捉更深层次的用户信念和推理。
*   **vs. 微调/偏好学习**: 微调和偏好学习通常需要大量数据，且难以适应用户偏好的动态变化。PRIME的记忆机制能够更灵活地更新和适应用户画像。

## Motivation

Large language model (LLM) personalization aims to align model outputs with individuals' unique preferences and opinions. While recent efforts have implemented various personalization methods, a unified theoretical framework that can systematically understand the drivers of effective personalization is still lacking.

We posit that the well-established cognitive dual-memory model can provide a robust theoretical foundation for LLM personalization. By mirroring episodic memory to historical user engagements and semantic memory to long-term, evolving user beliefs, we can capture a more comprehensive user profile.

## Contribution

*   **Unified Cognitive Dual-Memory Framework**: We are the first to systematically integrate the cognitive dual-memory model (episodic and semantic memory) into LLM personalization, proposing the PRIME framework.
*   **Personalized Thinking Capability**: We introduce a novel personalized thinking capability, inspired by the "slow thinking" strategy, to further enhance the model's reasoning and response quality.
*   **New Benchmark Dataset**: We developed a long-context personalization dataset using Change My View (CMV) from Reddit, addressing the limitations of existing benchmarks.
*   **Extensive Experimental Validation**: We conducted comprehensive experiments across both long- and short-context scenarios, validating PRIME's effectiveness in capturing dynamic personalization.

## Method

The PRIME framework achieves LLM personalization through the following core components:

### 1. Memory Instantiations

*   **Episodic Memory**: Mirrors historical user engagements, capturing specific events and contexts. This includes past conversations, preference choices, and other interactions.
*   **Semantic Memory**: Corresponds to long-term, evolving user beliefs and preferences. It is formed by abstracting and generalizing from episodic memory, representing more stable user characteristics.

### 2. Personalized Thought Process

*   **Slow Thinking Strategy**: Inspired by human slow thinking, PRIME performs an internal, multi-step reasoning process before generating a final response. This process leverages information from both episodic and semantic memory to deeply understand user intent and preferences, leading to more nuanced and personalized outputs.

## Comparison with Previous Work

*   **vs. Existing Personalization Methods**: Current methods often focus on a single form of memory (e.g., RAG or fine-tuning) and lack a unified theoretical framework. PRIME, by integrating cognitive dual-memory, provides a more comprehensive user profile.
*   **vs. RAG (Retrieval-Augmented Generation)**: RAG primarily relies on retrieving relevant documents, akin to a simpler form of episodic memory. PRIME builds upon this by adding semantic memory and a personalized thought process, enabling it to capture deeper user beliefs and reasoning.
*   **vs. Fine-tuning/Preference Learning**: Fine-tuning and preference learning typically require extensive data and struggle to adapt to dynamic user preference changes. PRIME's memory mechanisms allow for more flexible updating and adaptation of user profiles.
