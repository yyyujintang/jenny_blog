## Motivation

在基于大型语言模型的智能体中，记忆是实现用户个性化能力的关键，它能存储和利用用户信息。
现有研究虽然已将记忆应用于用户个性化，但通常侧重于偏好对齐和简单的问答任务。
然而，现实世界中的复杂任务往往需要对大量用户信息进行多跳推理，这对当前的记忆方法提出了严峻挑战。

## Method

### 1. 任务定义与数据集构建
- **多跳个性化推理任务**: 明确定义了该任务，要求智能体能对存储的个性化信息进行多步逻辑推理以回答复杂问题。
- **MPR数据集**: 构建了一个包含多跳推理和个性化信息的综合数据集，涵盖了多种复杂场景。

### 2. 显式记忆方法
- **RAG-based方法**: 利用检索增强生成（RAG）范式，通过显式检索相关记忆片段来辅助推理。
- **图神经网络（GNN）方法**: 将用户记忆构建成知识图谱，利用GNN进行多跳推理。

### 3. 隐式记忆方法
- **微调（Fine-tuning）**: 通过在个性化数据上微调大型语言模型，使其隐式地学习和存储用户偏好和信息。
- **上下文学习（In-context Learning）**: 将相关记忆片段直接作为上下文输入给LLM，使其在推理时利用这些信息。

### 4. HybridMem混合方法
- **结合显式与隐式**: HybridMem旨在结合显式记忆的精确检索能力和隐式记忆的泛化与推理能力。
- **动态记忆管理**: 提出了一种机制，根据任务需求和推理阶段动态地选择和融合显式或隐式记忆策略。
- **迭代推理与记忆更新**: 支持多轮迭代推理，并根据推理结果更新记忆，以适应更复杂的任务。

## Motivation

In large language model-based agents, memory is a critical capability for achieving personalization by storing and utilizing users' information.
While some previous studies have adopted memory to implement user personalization, they typically focus on preference alignment and simple question-answering.
However, in the real world, complex tasks often require multi-hop reasoning on a large amount of user information, which poses significant challenges for current memory approaches.

## Contribution

- **Multi-hop Personalized Reasoning Task**: We explicitly define this task to explore how different memory mechanisms perform in multi-hop reasoning over personalized information.
- **MPR Dataset and Unified Evaluation Framework**: We construct a novel dataset along with a unified evaluation framework to assess agent performance on multi-hop personalized reasoning.
- **Implementation and Evaluation of Memory Methods**: We implement various explicit and implicit memory methods and conduct comprehensive experiments to evaluate their performance, analyzing their strengths and weaknesses.
- **HybridMem Method**: We explore hybrid approaches that combine both explicit and implicit paradigms and propose the HybridMem method to address their limitations.
- **Project Release**: We release our project code and dataset to benefit the research community and foster further advancements in this area.

## Method

### 1. Task Definition and Dataset Construction
- **Multi-hop Personalized Reasoning Task**: We formally define this task, requiring agents to perform multi-step logical reasoning over stored personalized information to answer complex queries.
- **MPR Dataset**: We construct a comprehensive dataset featuring multi-hop reasoning and personalized information, covering diverse complex scenarios.

### 2. Explicit Memory Methods
- **RAG-based Approaches**: Utilizing the Retrieval-Augmented Generation (RAG) paradigm to explicitly retrieve relevant memory snippets to aid reasoning.
- **Graph Neural Network (GNN) Methods**: Structuring user memories as knowledge graphs and employing GNNs for multi-hop reasoning.

### 3. Implicit Memory Methods
- **Fine-tuning**: Fine-tuning large language models on personalized data to implicitly learn and store user preferences and information.
- **In-context Learning**: Providing relevant memory snippets directly as context to the LLM, enabling it to leverage this information during reasoning.

### 4. HybridMem Hybrid Approach
- **Combining Explicit and Implicit**: HybridMem aims to combine the precise retrieval capabilities of explicit memory with the generalization and reasoning power of implicit memory.
- **Dynamic Memory Management**: We propose a mechanism to dynamically select and integrate explicit or implicit memory strategies based on task requirements and reasoning stages.
- **Iterative Reasoning and Memory Update**: Our approach supports multi-round iterative reasoning and updates memory based on reasoning outcomes to adapt to more complex tasks.

## Comparison with Previous Work

- **vs. Traditional Personalization Methods**: Existing methods often focus on simple preference alignment or single-hop Q&A, whereas our work addresses the more challenging multi-hop complex reasoning.
- **vs. Existing RAG Approaches**: Most RAG systems do not adequately consider the multi-hop reasoning requirements for personalized information; our method extends and optimizes upon this.
- **vs. Existing Memory-Augmented LLMs**: Many memory-augmented LLMs focus on long-context processing or factual knowledge, while we specifically target multi-hop reasoning capabilities over user-personalized information.
- **vs. Existing Datasets**: Current datasets lack a comprehensive combination of multi-hop reasoning and personalized information, a gap filled by our MPR dataset.
