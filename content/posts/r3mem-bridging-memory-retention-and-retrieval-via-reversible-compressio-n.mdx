## Motivation

大型语言模型（LLMs）在实际应用中需要强大的记忆能力来提升性能。
现有的记忆解决方案面临权衡：显式记忆（基于外部存储）管理复杂且存储开销大。
隐式记忆（通过参数存储信息）则在可靠检索方面存在困难。

## Motivation

Memory plays a crucial role in enhancing the performance of Large Language Models (LLMs) when deployed in real-world applications.
Existing memory solutions face inherent trade-offs: explicit memory designs, relying on external storage, demand complex management and incur significant storage overhead.
Conversely, implicit memory designs, which store information via model parameters, struggle with reliable retrieval.

## Contribution

- **Reversible Context Compression**: Proposes R3Mem, a memory network that optimizes both information retention and retrieval through reversible context compression.
- **Virtual Memory Tokens**: Employs virtual memory tokens to compress and encode infinitely long histories.
- **Hierarchical Compression Strategy**: Utilizes a hierarchical compression strategy that refines information from document- to entity-level for improved assimilation across granularities.
- **Reversible Architecture**: Leverages a reversible architecture to reconstruct raw data by invoking the model backward with compressed information, enhancing retrieval.
- **Parameter-Efficient Fine-tuning**: Implemented via parameter-efficient fine-tuning, allowing seamless integration with any Transformer-based model.

## Method

### 1. Memory Encoding and Compression
1. **Virtual Memory Tokens**: R3Mem uses virtual memory tokens to compress and encode infinitely long historical contexts.
2. **Hierarchical Compression**: A hierarchical strategy refines information from document-level down to entity-level, ensuring better assimilation of information across different granularities.

### 2. Reversible Retrieval
1. **Backward Reconstruction**: R3Mem's reversible architecture allows for the reconstruction of raw data by invoking the model backward using the compressed information.
2. **Optimized Retrieval**: This reversibility ensures reliable retrieval of necessary information during the retrieval phase.

### 3. Model Integration
1. **Parameter-Efficient Fine-tuning**: R3Mem is implemented via parameter-efficient fine-tuning (PEFT), enabling seamless integration with any Transformer-based LLM.

## Comparison with Previous Work

- **vs. Explicit Memory Designs**: Existing explicit memory solutions (e.g., external storage) require complex management and high storage overhead, which R3Mem mitigates through compression.
- **vs. Implicit Memory Designs**: Current implicit memory designs (storing information via parameters) struggle with reliable retrieval; R3Mem's reversible architecture significantly improves retrieval capabilities.
- **vs. Conventional Memory Modules**: R3Mem significantly outperforms conventional memory modules in long-horizon interaction tasks such as conversational agents.
