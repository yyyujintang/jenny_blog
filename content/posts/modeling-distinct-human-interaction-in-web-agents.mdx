## Motivation

尽管自主web agent发展迅速，但在任务执行过程中，人类的参与对于塑造偏好和纠正agent行为仍然至关重要。
然而，当前的agent系统缺乏对人类何时以及为何干预的原则性理解，经常在关键决策点自主前进或请求不必要的确认。
这导致了协作效率低下和用户体验不佳。

## Contribution

- **人类干预建模任务**: 首次引入了建模人类干预的任务，以支持协作式web任务执行。
- **CowCorpus数据集**: 构建并发布了CowCorpus数据集，包含400条真实用户web导航轨迹和超过4,200个交错的人类与agent动作。
- **四种用户交互模式**: 识别并定义了用户与agent交互的四种独特模式：hands-off supervision（放手监督）、hands-on oversight（动手监督）、collaborative task-solving（协作解决任务）和full user takeover（完全用户接管）。
- **干预预测模型**: 训练语言模型（LMs）根据用户交互风格预测用户何时可能干预，将干预预测准确率比基线LMs提高了61.4-63.4%。
- **用户研究验证**: 将这些干预感知模型部署到实时web导航agent中，并通过用户研究发现用户对agent有用性的评价提高了26.5%。

## Method

1.  **数据收集**: 收集真实用户在web导航任务中的交互轨迹，构建CowCorpus数据集。
2.  **交互模式识别**: 分析数据集，识别并定义四种主要的用户与agent交互模式：
    *   **Hands-off supervision**: 用户仅进行高层级监督，很少直接干预。
    *   **Hands-on oversight**: 用户偶尔介入纠正或引导agent。
    *   **Collaborative task-solving**: 用户与agent交替执行任务步骤，共同完成目标。
    *   **Full user takeover**: 用户完全接管任务，agent退居次要地位。
3.  **特征工程**: 从交互轨迹中提取特征，包括agent的动作历史、用户的干预历史、任务上下文等，以表征用户的交互风格。
4.  **LM训练**: 使用提取的特征训练语言模型，预测在给定当前状态和用户交互风格下，用户在下一步进行干预的可能性。
5.  **模型部署与评估**: 将训练好的干预预测模型集成到实时web导航agent中，并在用户研究中评估其对用户体验和agent有用性的影响。

## Motivation

Despite rapid progress in autonomous web agents, human involvement remains essential for shaping preferences and correcting agent behavior as tasks unfold.
However, current agentic systems lack a principled understanding of when and why humans intervene, often proceeding autonomously past critical decision points or requesting unnecessary confirmation.
This leads to inefficient collaboration and a suboptimal user experience.

## Contribution

-   **Human Intervention Modeling Task**: Introduces the novel task of modeling human intervention to support collaborative web task execution.
-   **CowCorpus Dataset**: Collects and releases CowCorpus, a dataset of 400 real-user web navigation trajectories containing over 4,200 interleaved human and agent actions.
-   **Four Distinct User Interaction Patterns**: Identifies and characterizes four distinct patterns of user interaction with agents: hands-off supervision, hands-on oversight, collaborative task-solving, and full user takeover.
-   **Intervention Prediction Models**: Trains language models (LMs) to anticipate when users are likely to intervene based on their interaction styles, yielding a 61.4-63.4% improvement in intervention prediction accuracy over base LMs.
-   **User Study Validation**: Deploys these intervention-aware models in live web navigation agents and evaluates them in a user study, finding a 26.5% increase in user-rated agent usefulness.

## Method

1.  **Data Collection**: Gathers real-user interaction trajectories during web navigation tasks to construct the CowCorpus dataset.
2.  **Interaction Pattern Identification**: Analyzes the dataset to identify and define four primary user-agent interaction patterns:
    *   **Hands-off supervision**: Users provide high-level oversight with minimal direct intervention.
    *   **Hands-on oversight**: Users occasionally intervene to correct or guide the agent.
    *   **Collaborative task-solving**: Users and agents alternate in performing task steps, working together towards a goal.
    *   **Full user takeover**: Users completely assume control of the task, with the agent taking a secondary role.
3.  **Feature Engineering**: Extracts features from interaction trajectories, including agent action history, user intervention history, and task context, to characterize user interaction styles.
4.  **LM Training**: Trains language models using the extracted features to predict the likelihood of user intervention at the next step, given the current state and user interaction style.
5.  **Model Deployment and Evaluation**: Integrates the trained intervention prediction models into live web navigation agents and assesses their impact on user experience and agent usefulness through a user study.

## Comparison with Previous Work

-   **vs. Traditional Autonomous Agents**: Traditional agents prioritize autonomy and lack an understanding of or adaptability to human intervention, potentially making autonomous decisions at critical junctures or requesting unnecessary confirmations. This work enhances collaboration by predicting intervention.
-   **vs. Task-Context-Only Prediction**: Some existing works might predict intervention solely based on task complexity or current state. This paper further incorporates the user's "interaction style," providing more nuanced and personalized predictions.
-   **vs. Research Lacking Real-User Data**: Many studies on human-AI collaboration rely on simulated data or small-scale experiments. This work provides robust evidence based on real-user behavior through the large-scale CowCorpus dataset.
