## Motivation

当前的AI Agent在处理复杂、长期的多模态任务时，往往受限于其记忆能力，尤其是在视觉信息方面。
传统的Agent记忆系统通常偏向于文本信息，对视觉记忆的处理不够精细和动态。
Agent需要一个能够持续学习、演化并精炼视觉记忆的机制，以更好地理解和应对真实世界的复杂视觉场景。

## Contribution

- **视觉记忆的grow-and-refine机制**: 提出一种动态增长和精炼视觉语义记忆的方法，使其能随时间演化。
- **多模态语义记忆系统**: 整合视觉与语言信息，构建统一的多模态记忆表示，支持更全面的理解。
- **Agentic Learner框架**: 将记忆机制融入Agent的学习循环中，使其能够自主地从经验中学习和改进。
- **提升复杂多模态推理能力**: 显著提高了Agent在需要长期视觉记忆和推理的多模态任务上的表现。

## Method

ViLoMem的核心在于其对视觉记忆的独特处理方式，通过以下步骤实现：

1.  **视觉信息编码与抽取**:
    *   Agent首先接收多模态输入（图像、文本查询），并利用预训练的视觉编码器抽取图像特征。
    *   通过AnalyzeGenerate^V模块，从视觉输入中提取关键的视觉实体(e^V_i)和生成新的视觉概念(g^V_i)。
2.  **视觉记忆的增长 (Grow)**:
    *   新生成的视觉概念(g^V_i)被整合到Agent的视觉记忆库M^V中。
    *   记忆库会根据新的信息进行扩展，捕获Agent与环境交互中遇到的新视觉元素和关系。
3.  **视觉记忆的精炼 (Refine)**:
    *   通过Merge^V机制，系统会周期性地对现有视觉记忆进行合并、去重和抽象。
    *   这有助于消除冗余信息，强化重要概念，并形成更高级别的视觉语义表示。
4.  **多模态语义记忆整合**:
    *   视觉记忆(M^V)与语言记忆(M^L)通过共享的语义空间进行交互和融合。
    *   Agent在进行推理时，能够同时检索和利用视觉与语言记忆，形成全面的情境理解。
5.  **Agentic Learning循环**:
    *   Agent根据当前任务和记忆进行决策，执行动作，并从环境反馈中学习。
    *   学习过程会进一步驱动视觉记忆的增长和精炼，形成一个闭环的自我改进机制。

## Motivation

Current AI agents often struggle with complex, long-term multimodal tasks due to limitations in their memory capabilities, especially concerning visual information.
Traditional agent memory systems typically prioritize textual information, lacking the nuanced and dynamic processing required for visual memory.
Agents need a mechanism that can continuously learn, evolve, and refine visual memories to better understand and respond to the complexities of real-world visual scenarios.

## Contribution

- **Grow-and-refine mechanism for visual memory**: Proposes a method for dynamically growing and refining visual semantic memory, allowing it to evolve over time.
- **Multimodal semantic memory system**: Integrates visual and linguistic information to construct a unified multimodal memory representation, supporting more comprehensive understanding.
- **Agentic Learner framework**: Embeds the memory mechanism within the agent's learning loop, enabling autonomous learning and improvement from experience.
- **Enhanced complex multimodal reasoning**: Significantly improves agent performance on multimodal tasks requiring long-term visual memory and reasoning.

## Method

ViLoMem's core lies in its unique approach to visual memory processing, implemented through the following steps:

1.  **Visual Information Encoding and Extraction**:
    *   The agent receives multimodal input (images, text queries) and uses a pre-trained visual encoder to extract image features.
    *   Key visual entities (e^V_i) are extracted and new visual concepts (g^V_i) are generated from the visual input using the AnalyzeGenerate^V module.
2.  **Visual Memory Growth (Grow)**:
    *   Newly generated visual concepts (g^V_i) are integrated into the agent's visual memory bank M^V.
    *   The memory bank expands with new information, capturing novel visual elements and relationships encountered during the agent's interaction with the environment.
3.  **Visual Memory Refinement (Refine)**:
    *   Through a Merge^V mechanism, the system periodically merges, de-duplicates, and abstracts existing visual memories.
    *   This helps eliminate redundancy, reinforce important concepts, and form higher-level visual semantic representations.
4.  **Multimodal Semantic Memory Integration**:
    *   Visual memory (M^V) interacts and fuses with linguistic memory (M^L) through a shared semantic space.
    *   During reasoning, the agent can simultaneously retrieve and utilize both visual and linguistic memories for comprehensive contextual understanding.
5.  **Agentic Learning Loop**:
    *   The agent makes decisions based on the current task and memory, executes actions, and learns from environmental feedback.
    *   This learning process further drives the growth and refinement of visual memory, forming a closed-loop self-improvement mechanism.
