## Motivation

在多样化、动态的环境中运行的机器人操作系统，需要具备多任务交互、泛化到未见场景以及空间记忆这三项关键能力。

尽管机器人操作领域已取得显著进展，但现有方法在泛化到复杂环境变化和处理依赖记忆的任务方面仍有不足。

为了弥合这一差距，我们提出了SAM2Act，一个结合视觉基础模型与记忆架构的机器人策略。

## Contribution

- **SAM2Act策略**: 提出一个基于多视图机器人Transformer的策略SAM2Act，它利用多分辨率上采样和来自大规模视觉基础模型的视觉表示，在RLBench基准测试的18项任务中实现了86.8%的SOTA平均成功率。
- **鲁棒泛化能力**: SAM2Act在The Colosseum基准测试中展示了强大的泛化能力，在多样化的环境扰动下性能差距仅为4.3%。
- **SAM2Act+记忆架构**: 提出受SAM2启发的记忆增强架构SAM2Act+，它包含一个记忆库、一个编码器和一个注意力机制，以增强空间记忆能力。
- **MemoryBench基准**: 引入了一个新的基准测试MemoryBench，专门用于评估机器人操作中的空间记忆和动作召回能力。
- **记忆任务SOTA**: SAM2Act+在MemoryBench的记忆相关任务中实现了94.3%的平均成功率，显著优于现有方法。

## Method

### SAM2Act: 视觉基础模型集成策略

1.  **多视图机器人Transformer**: SAM2Act是一个基于Transformer的策略，能够处理多视图输入。
2.  **多分辨率上采样**: 利用多分辨率上采样技术，从视觉基础模型中提取丰富的视觉表示。
3.  **大规模视觉基础模型**: 集成来自大型视觉基础模型的视觉特征，以增强对环境的理解和泛化能力。

### SAM2Act+: 记忆增强架构

1.  **记忆库 (Memory Bank)**: 存储过去的关键状态和动作信息，作为机器人的长期记忆。
2.  **编码器 (Encoder)**: 处理当前感知输入和记忆库中的信息，生成上下文相关的表示。
3.  **注意力机制 (Attention Mechanism)**: 允许模型选择性地关注记忆库中最相关的历史信息，以指导当前决策。

## Motivation

Robotic manipulation systems operating in diverse, dynamic environments must exhibit three critical abilities: multitask interaction, generalization to unseen scenarios, and spatial memory.

While significant progress has been made in robotic manipulation, existing approaches often fall short in generalization to complex environmental variations and addressing memory-dependent tasks.

To bridge this gap, we introduce SAM2Act, a robotic policy that leverages visual foundation models and a memory architecture.

## Contribution

- **SAM2Act Policy**: We introduce SAM2Act, a multi-view robotic transformer-based policy that leverages multi-resolution upsampling with visual representations from large-scale foundation models, achieving a state-of-the-art average success rate of 86.8% across 18 tasks in the RLBench benchmark.
- **Robust Generalization**: SAM2Act demonstrates robust generalization on The Colosseum benchmark, with only a 4.3% performance gap under diverse environmental perturbations.
- **SAM2Act+ Memory Architecture**: We propose SAM2Act+, a memory-based architecture inspired by SAM2, which incorporates a memory bank, an encoder, and an attention mechanism to enhance spatial memory.
- **MemoryBench Benchmark**: We introduce MemoryBench, a novel benchmark designed to assess spatial memory and action recall in robotic manipulation.
- **SOTA on Memory Tasks**: SAM2Act+ achieves an average success rate of 94.3% on memory-based tasks in MemoryBench, significantly outperforming existing approaches.

## Method

### SAM2Act: Visual Foundation Model Integration Policy

1.  **Multi-view Robotic Transformer**: SAM2Act is a Transformer-based policy capable of processing multi-view inputs.
2.  **Multi-resolution Upsampling**: It utilizes multi-resolution upsampling techniques to extract rich visual representations from visual foundation models.
3.  **Large-scale Visual Foundation Model**: Integrates visual features from large-scale visual foundation models to enhance environmental understanding and generalization capabilities.

### SAM2Act+: Memory-Enhanced Architecture

1.  **Memory Bank**: Stores critical past states and action information, serving as the robot's long-term memory.
2.  **Encoder**: Processes current perceptual input and information from the memory bank to generate context-aware representations.
3.  **Attention Mechanism**: Allows the model to selectively focus on the most relevant historical information in the memory bank to guide current decision-making.

## Comparison with Previous Work

- **vs. Existing Robotic Manipulation Approaches**: Current methods often struggle with generalization in complex environments and memory-dependent tasks, whereas SAM2Act significantly improves generalization by integrating visual foundation models and multi-resolution upsampling.
- **vs. RVT-2**: SAM2Act surpasses RVT-2 in multi-task performance on RLBench, achieving a higher average success rate, demonstrating its superiority in complex manipulation tasks.
- **vs. Methods Lacking Explicit Memory Mechanisms**: SAM2Act+, by introducing an explicit memory architecture (memory bank, encoder, attention mechanism), significantly outperforms baseline methods without memory mechanisms on MemoryBench's memory-related tasks, addressing their shortcomings in spatial memory and action recall.
- **vs. SAM-E**: SAM2Act performs better than SAM-E on RLBench, showing stronger robustness particularly in certain specific tasks.
