## Motivation

尽管视觉语言模型（VLMs）取得了显著成功，但在处理复杂视觉任务时，它们常因“视觉处理瓶颈”而受限。

这种瓶颈表现为在长时间生成过程中，模型容易失去对视觉证据的 grounding，并缺乏上下文感知的视觉经验。

本文从人类认知记忆理论中汲取灵感，该理论区分了以视觉为主的短期记忆和以语义为主的长期记忆。

## Contribution

- **认知对齐的记忆框架（cognitively-aligned framework）**: 提出了VisMem，一个受人类认知记忆启发的框架，旨在解决VLM的视觉处理瓶颈。
- **动态潜在视觉记忆（dynamic latent vision memories）**: 为VLM配备了动态的潜在视觉记忆，包括用于精细感知保留的短期模块和用于抽象语义整合的长期模块。
- **无缝记忆调用（seamless memory invocation）**: 记忆模块在推理过程中被无缝调用，使VLM在思考和生成过程中都能保持感知保真度和语义一致性。
- **性能显著提升（significant performance boost）**: 在多样的视觉基准测试中，VisMem相对于 vanilla 模型平均性能提升了11.0%，并超越了所有现有方法。

## Method

VisMem框架为VLM配备了动态潜在视觉记忆，其核心在于模拟人类的短期和长期记忆机制。

1.  **短期记忆模块（Short-term Module）**:
    *   专注于精细的感知信息保留。
    *   在处理当前视觉输入时，捕获并存储高分辨率、细节丰富的视觉特征。
    *   这有助于模型在短期内维持对视觉证据的精确 grounding。

2.  **长期记忆模块（Long-term Module）**:
    *   负责抽象语义的整合和存储。
    *   从历史交互和经验中提取并巩固更高级别的语义概念和模式。
    *   这使得模型能够建立上下文感知的视觉经验，并在长时间生成中保持语义一致性。

3.  **记忆无缝调用（Seamless Memory Invocation）**:
    *   在VLM的推理和生成过程中，这两个记忆模块被动态地、无缝地调用。
    *   短期记忆提供即时的感知细节，而长期记忆提供宏观的语义上下文。
    *   这种协同作用确保了模型在处理复杂视觉任务时，既能保持对视觉细节的忠实度，又能维持整体的语义连贯性。

## Motivation

Despite the remarkable success of Vision-Language Models (VLMs), their performance on a range of complex visual tasks is often hindered by a "visual processing bottleneck."

This bottleneck manifests as a propensity to lose grounding in visual evidence and exhibit a deficit in contextualized visual experience during prolonged generation.

Drawing inspiration from human cognitive memory theory, which distinguishes short-term visually-dominant memory and long-term semantically-dominant memory, this paper proposes a new framework.

## Contribution

- **Cognitively-Aligned Framework**: Introduces VisMem, a framework inspired by human cognitive memory theory, designed to address the visual processing bottleneck in VLMs.
- **Dynamic Latent Vision Memories**: Equips VLMs with dynamic latent vision memories, featuring a short-term module for fine-grained perceptual retention and a long-term module for abstract semantic consolidation.
- **Seamless Memory Invocation**: These memory modules are seamlessly invoked during inference, enabling VLMs to maintain both perceptual fidelity and semantic consistency across thinking and generation.
- **Significant Performance Boost**: VisMem delivers a substantial average performance boost of 11.0% relative to the vanilla model across diverse visual benchmarks, outperforming all counterparts.

## Method

The VisMem framework equips VLMs with dynamic latent vision memories, centrally mimicking human short-term and long-term memory mechanisms.

1.  **Short-term Module**:
    *   Focuses on fine-grained perceptual retention.
    *   Captures and stores high-resolution, detail-rich visual features when processing current visual input.
    *   This helps the model maintain precise grounding in visual evidence over short durations.

2.  **Long-term Module**:
    *   Responsible for the consolidation and storage of abstract semantics.
    *   Extracts and solidifies higher-level semantic concepts and patterns from historical interactions and experiences.
    *   This enables the model to build contextualized visual experiences and maintain semantic consistency during prolonged generation.

3.  **Seamless Memory Invocation**:
    *   During the VLM's inference and generation processes, these two memory modules are dynamically and seamlessly invoked.
    *   The short-term memory provides immediate perceptual details, while the long-term memory offers macroscopic semantic context.
    *   This synergistic interaction ensures that the model maintains both fidelity to visual details and overall semantic coherence when handling complex visual tasks.

## Comparison with Previous Work

- **vs. Vanilla VLMs**: Traditional VLMs suffer from a "visual processing bottleneck" during prolonged generation, leading to a loss of visual grounding and a lack of contextual visual experience. VisMem directly addresses this core issue by introducing dynamic memory modules.
- **vs. Other Memory Enhancement Methods**: VisMem's unique aspect lies in its cognitively-aligned design, distinguishing between short-term visually-dominant memory and long-term semantically-dominant memory. This differs from most existing general memory enhancement methods, which may not have such a fine-grained visual-semantic memory distinction.
- **vs. RAG-based VLMs**: While RAG (Retrieval-Augmented Generation) can provide external knowledge to VLMs, VisMem focuses on internal visual memory and contextual integration within the model, rather than solely external knowledge retrieval.
