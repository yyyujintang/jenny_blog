## Motivation

现有的多模态基准测试主要评估单轮视觉推理或特定工具技能。
这些基准未能充分捕捉现实世界代理所需的真实性、视觉细节和长序列工具使用。
实际应用中的多模态代理需要解决基于视觉证据的多步骤工作流程，例如通过图片诊断设备问题或根据地图规划行程。

## Contribution

- **AgentVista基准测试**: 引入了一个用于通用多模态代理的新基准，涵盖7个类别25个子领域。
- **真实且细节丰富的视觉场景**: 任务设计贴近现实，包含大量视觉细节。
- **自然混合工具使用**: 任务要求代理在多模态环境下进行长序列工具交互，包括web search, image search, page navigation, 以及基于代码的图像处理和通用编程操作。
- **揭示现有模型差距**: 对最先进模型进行全面评估，发现它们在长序列多模态工具使用方面存在显著差距。

## Method

AgentVista基准测试通过以下方式构建：
1.  **场景设计**: 涵盖25个子领域，分布在7个主要类别中，模拟真实世界的复杂任务。
2.  **任务类型**: 要求代理执行多步骤工作流程，这些流程需要结合视觉证据和多种工具。
3.  **工具集**: 包含web search, image search, page navigation, 以及用于图像处理和通用编程的code-based operations。
4.  **评估指标**: 衡量代理在长序列工具交互下的准确性，以及处理视觉细微差别的能力。

- **vs. TIR-Bench, Agent-X, MMSearch-Plus, BrowseComp-VL, VisualToolBench**: AgentVista在工具使用广度（同时支持Visual Operations, Visual Search, Text Search, Code Execution）和任务复杂性（平均工具调用轮次12.67次，远高于其他基准的2.92-4.6次）方面显著超越现有基准。

- **vs. 特定工具技能基准**: AgentVista评估的是通用多模态代理，要求代理能够灵活组合多种工具来解决复杂问题，而非仅测试单一工具的使用能力。

## Motivation

Existing multimodal benchmarks primarily evaluate single-turn visual reasoning or specific tool skills.
These benchmarks fail to fully capture the realism, visual subtlety, and long-horizon tool use required by practical agents.
Real-world multimodal agents need to solve multi-step workflows grounded in visual evidence, such as troubleshooting a device from a photo or planning a trip using a map.

## Contribution

- **AgentVista Benchmark**: Introduction of a new benchmark for generalist multimodal agents, spanning 25 sub-domains across 7 categories.
- **Realistic and Detail-Rich Visual Scenarios**: Tasks are designed to be realistic and incorporate rich visual details.
- **Natural Hybrid Tool Use**: Tasks require long-horizon tool interactions across modalities, including web search, image search, page navigation, and code-based operations for both image processing and general programming.
- **Exposure of Model Gaps**: Comprehensive evaluation of state-of-the-art models reveals significant gaps in their ability to carry out long-horizon multimodal tool use.

## Method

The AgentVista benchmark is constructed by:
1.  **Scenario Design**: Covering 25 sub-domains across 7 primary categories, simulating complex real-world tasks.
2.  **Task Types**: Requiring agents to execute multi-step workflows that necessitate combining visual evidence with various tools.
3.  **Tool Set**: Incorporating web search, image search, page navigation, and code-based operations for image processing and general programming.
4.  **Evaluation Metrics**: Measuring agent accuracy in long-horizon tool interactions and their ability to handle visual subtleties.

## Comparison with Previous Work

- **vs. TIR-Bench, Agent-X, MMSearch-Plus, BrowseComp-VL, VisualToolBench**: AgentVista significantly surpasses existing benchmarks in terms of tool usage breadth (simultaneously supporting Visual Operations, Visual Search, Text Search, Code Execution) and task complexity (average tool-calling turns of 12.67, much higher than 2.92-4.6 for others).
- **vs. Existing Single-Turn Visual Reasoning Benchmarks**: AgentVista focuses on multi-step tasks requiring long-horizon tool interactions, rather than simple single-turn visual question answering.
- **vs. Specific Tool Skill Benchmarks**: AgentVista evaluates generalist multimodal agents, demanding flexible combination of multiple tools to solve complex problems, not just testing the use of a single tool.
