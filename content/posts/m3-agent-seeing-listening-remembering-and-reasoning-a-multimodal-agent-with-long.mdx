## Motivation

当前的AI智能体在处理复杂、长期的多模态任务时，往往缺乏有效的长时记忆机制。这限制了它们像人类一样积累世界知识、进行多轮推理的能力。

现有方法多侧重于短期或单模态记忆，难以应对需要跨时间、跨模态信息整合的复杂场景。

本研究旨在弥补这一空白，构建一个能像人类一样“看、听、记忆、推理”的多模态智能体。

## Contribution

- **M3-Agent框架**: 提出了一个新颖的多模态智能体框架，能够处理实时视听输入，并构建和更新情景记忆（episodic memory）和语义记忆（semantic memory）。
- **实体中心、多模态记忆组织**: 设计了一种实体中心（entity-centric）的多模态记忆组织方式，实现了对环境更深层次、更一致的理解。
- **M3-Bench基准**: 开发了一个新的长视频问答基准M3-Bench，包含M3-Bench-robot和M3-Bench-web，用于评估多模态智能体在记忆有效性和基于记忆推理方面的能力。
- **强化学习训练**: 通过强化学习（RL）训练M3-Agent，使其在多模态记忆和推理任务中表现出色。

## Method

M3-Agent框架设计旨在模拟人类的记忆和推理过程，主要包括以下几个核心组件：

### 1. 感知与记忆构建
- **实时输入处理**: M3-Agent能够实时处理视觉和听觉输入流。
- **记忆编码**: 将感知到的信息编码为多模态记忆节点，这些节点包含内容、嵌入（embedding）、权重和元数据等属性。
- **记忆类型**: 构建两种主要记忆类型：
    - **情景记忆 (Episodic Memory)**: 记录特定事件和经历，按时间顺序组织。
    - **语义记忆 (Semantic Memory)**: 存储抽象的世界知识和概念，通过整合情景记忆形成。
- **实体中心组织**: 记忆以实体为中心进行组织，使得智能体能围绕特定对象、人物或地点积累知识，实现更连贯的理解。

### 2. 记忆检索
- **多模态检索**: 当接收到指令时，M3-Agent能够根据查询（文本、图像或音频）从长时记忆中检索相关的记忆节点或记忆片段（memory clips）。
- **相似性匹配**: 利用记忆节点的embedding进行相似性匹配，高效地找到相关信息。

### 3. 多轮推理与任务完成
- **自主推理**: M3-Agent能够根据指令自主进行多轮推理。
- **记忆辅助**: 在推理过程中，智能体主动检索相关记忆来辅助决策和问题解决。
- **任务执行**: 结合检索到的记忆和推理结果，完成复杂任务。

## Comparison with Previous Work

- **vs. 传统RAG方法**: 传统的RAG（Retrieval-Augmented Generation）通常依赖于文本检索，而M3-Agent扩展到多模态，能够从视觉和听觉信息中检索，并构建更复杂的实体中心记忆结构。
- **vs. 短期记忆模型**: 许多现有模型主要关注短期上下文窗口，M3-Agent通过构建情景记忆和语义记忆，实现了真正意义上的长时记忆，能够处理长时间跨度的信息。
- **vs. 仅依赖Prompting的智能体**: 像Gemini-1.5-pro和GPT-4o等Prompting智能体虽然强大，但在需要深度记忆和多轮推理的长视频任务中，M3-Agent通过其结构化的长时记忆和RL训练展现出更优的性能。
- **vs. 现有视频理解基准**: M3-Bench专注于评估多模态智能体的记忆有效性和基于记忆的推理能力，涵盖了多证据推理、多跳推理、跨模态推理、人物理解和通用知识提取等复杂问答类型，比现有基准更全面地测试了智能体的长时记忆能力。

## Motivation

Current AI agents often lack effective long-term memory mechanisms when dealing with complex, prolonged multimodal tasks. This limitation hinders their ability to accumulate world knowledge and perform multi-turn reasoning like humans.

Existing methods mostly focus on short-term or unimodal memory, struggling to handle complex scenarios that require integrating information across time and modalities.

This research aims to bridge this gap by building a multimodal agent that can "see, listen, remember, and reason" in a human-like manner.

## Contribution

- **M3-Agent Framework**: Introduced a novel multimodal agent framework capable of processing real-time visual and auditory inputs to build and update episodic and semantic memories.
- **Entity-Centric, Multimodal Memory Organization**: Designed an entity-centric, multimodal memory organization that enables deeper and more consistent understanding of the environment.
- **M3-Bench Benchmark**: Developed a new long-video question answering benchmark, M3-Bench (comprising M3-Bench-robot and M3-Bench-web), to evaluate multimodal agents' memory effectiveness and memory-based reasoning.
- **Reinforcement Learning Training**: Trained M3-Agent via reinforcement learning (RL), demonstrating superior performance in multimodal memory and reasoning tasks.

## Method

The M3-Agent framework is designed to simulate human memory and reasoning processes, primarily comprising the following core components:

### 1. Perception and Memory Construction
- **Real-time Input Processing**: M3-Agent can process real-time visual and auditory input streams.
- **Memory Encoding**: Perceived information is encoded into multimodal memory nodes, which include attributes like content, embedding, weight, and metadata.
- **Memory Types**: Two main types of memory are constructed:
    - **Episodic Memory**: Records specific events and experiences, organized chronologically.
    - **Semantic Memory**: Stores abstract world knowledge and concepts, formed by integrating episodic memories.
- **Entity-Centric Organization**: Memory is organized around entities, allowing the agent to accumulate knowledge about specific objects, people, or locations for more coherent understanding.

### 2. Memory Retrieval
- **Multimodal Retrieval**: Upon receiving an instruction, M3-Agent can retrieve relevant memory nodes or memory clips from its long-term memory based on the query (text, image, or audio).
- **Similarity Matching**: Utilizes the embeddings of memory nodes for similarity matching to efficiently find relevant information.

### 3. Multi-turn Reasoning and Task Completion
- **Autonomous Reasoning**: M3-Agent can autonomously perform multi-turn reasoning based on instructions.
- **Memory-Assisted**: During reasoning, the agent actively retrieves relevant memories to aid decision-making and problem-solving.
- **Task Execution**: Combines retrieved memories and reasoning results to complete complex tasks.

## Comparison with Previous Work

- **vs. Traditional RAG methods**: Traditional RAG (Retrieval-Augmented Generation) typically relies on text retrieval, whereas M3-Agent extends to multimodal retrieval from visual and auditory information, building more complex entity-centric memory structures.
- **vs. Short-term memory models**: Many existing models primarily focus on short-term context windows. M3-Agent achieves true long-term memory by constructing episodic and semantic memories, capable of processing information over extended durations.
- **vs. Prompting-only agents**: While powerful, prompting agents like Gemini-1.5-pro and GPT-4o show limitations in long-video tasks requiring deep memory and multi-turn reasoning. M3-Agent, with its structured long-term memory and RL training, demonstrates superior performance.
- **vs. Existing video understanding benchmarks**: M3-Bench specifically evaluates multimodal agents' memory effectiveness and memory-based reasoning capabilities, covering complex QA types such as multi-evidence reasoning, multi-hop reasoning, cross-modal reasoning, person understanding, and general knowledge extraction, offering a more comprehensive test of long-term memory.
