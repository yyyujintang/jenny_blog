## Motivation

人类智能的一个关键特征是能够持续获取、组织和利用知识，这也是AI系统要充分发挥潜力所必须近似的能力。在LLM的持续学习方面存在挑战，因此RAG已成为引入新信息的主要方式。然而，RAG对向量检索的依赖限制了其模仿人类长期记忆动态和互联性质的能力。

最近的RAG方法通过知识图谱等各种结构增强了向量嵌入，以解决部分不足，例如意义构建（sense-making）和联想性（associativity）。然而，它们在更基本的事实记忆任务上的表现却显著低于标准RAG。本研究旨在解决这种意外的性能下降。

## Contribution

HippoRAG 2是一个新颖的框架，它在事实、意义构建和联想记忆任务上全面超越了标准RAG。它在联想记忆任务上比最先进的嵌入模型提高了7%，同时在事实知识和意义构建记忆能力方面也表现出色。

该工作为LLM的非参数持续学习铺平了道路。HippoRAG 2通过解决现有结构增强RAG方法在基本事实记忆任务上的性能下降问题，实现了更接近人类长期记忆的有效性。

## Method

HippoRAG 2建立在HippoRAG中使用的Personalized PageRank算法之上，并对其进行了增强。

主要改进包括：
*   **深度passage整合（deeper passage integration）**：这可能涉及更细致地处理检索到的文本片段，使其与现有知识结构更紧密地结合。
*   **更有效的在线LLM使用（more effective online use of an LLM）**：这意味着LLM在检索和生成过程中扮演了更积极、更智能的角色，可能用于动态地构建、更新或查询记忆结构。
*   这种组合旨在将RAG系统推向更接近人类长期记忆的有效性，从而实现更全面的记忆能力。

## Motivation

The continuous acquisition, organization, and leveraging of knowledge are hallmarks of human intelligence that AI systems must approximate to unlock their full potential. Given the challenges in continual learning with large language models (LLMs), retrieval-augmented generation (RAG) has emerged as the dominant method for introducing new information. However, RAG's reliance on vector retrieval inherently limits its ability to mimic the dynamic and interconnected nature of human long-term memory.

Recent RAG approaches have attempted to augment vector embeddings with various structures, such as knowledge graphs, to address gaps in sense-making and associativity. Nevertheless, these methods often exhibit a considerable drop in performance on more basic factual memory tasks compared to standard RAG. This research aims to resolve this unintended deterioration in performance.

## Contribution

HippoRAG 2 is a novel framework that comprehensively outperforms standard RAG across factual, sense-making, and associative memory tasks. It achieves a 7% improvement in associative memory tasks over the state-of-the-art embedding model, while also demonstrating superior factual knowledge and sense-making memory capabilities.

This work paves the way for non-parametric continual learning for LLMs. By addressing the performance degradation of existing structure-augmented RAG methods on fundamental factual memory tasks, HippoRAG 2 brings RAG systems closer to the effectiveness of human long-term memory.

## Method

HippoRAG 2 builds upon the Personalized PageRank algorithm, which was a core component of its predecessor, HippoRAG, and introduces significant enhancements.

Key methodological improvements include:
*   **Deeper passage integration**: This likely involves more sophisticated mechanisms for incorporating retrieved text passages into the existing knowledge structure, ensuring a richer and more interconnected representation of information.
*   **More effective online use of an LLM**: The LLM plays a more active and intelligent role throughout the retrieval and generation process. This could involve dynamic knowledge graph construction, real-time query refinement, or adaptive memory updating based on the LLM's understanding.
*   This combination is designed to push the RAG system closer to the effectiveness of human long-term memory, enabling a more comprehensive range of memory capabilities.

## Comparison with Previous Work

HippoRAG 2 is benchmarked against a diverse set of RAG methodologies, including simple baselines (e.g., BM25, Contriever), large embedding models (e.g., GTE-Qwen2-7B-Instruct, NV-Embed-v2), and other structure-augmented RAG approaches (e.g., RAPTOR, GraphRAG, LightRAG, and the original HippoRAG).

A key distinction of HippoRAG 2 is its ability to significantly improve performance on sense-making and associative memory tasks while maintaining or even enhancing factual memory capabilities, unlike previous structure-augmented RAG methods that often suffered performance drops on factual tasks. HippoRAG 2 successfully navigates this trade-off, offering a more balanced and robust memory system.
