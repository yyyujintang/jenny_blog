当前AI智能体在处理复杂、多步骤任务时，常因缺乏有效的记忆机制而表现不佳。它们难以整合历史信息、学习长期模式，或在不同情境间迁移知识。

现有模型通常依赖于短期上下文窗口或简单的RAG机制，这限制了它们对过去经验的深度利用和适应新环境的能力。

## Method

### 1. 记忆编码与存储

- **短期记忆 (Working Memory)**: 使用Transformer的注意力机制来捕获即时上下文信息，并将其编码为向量表示。
- **情景记忆 (Episodic Memory)**: 存储智能体的具体经验序列（观察、动作、奖励），采用图结构或时间序列数据库进行组织。
- **语义记忆 (Semantic Memory)**: 建立一个知识图谱 (KG) 来存储事实性知识和概念关系，通过嵌入技术进行表示。

### 2. 记忆检索机制

- **上下文感知检索**: 根据当前任务和短期记忆内容，动态调整检索策略，从情景记忆和语义记忆中并行检索相关信息。
- **RAG增强**: 将检索到的记忆片段与当前输入结合，通过RAG机制喂给大型语言模型，以生成更准确、信息更丰富的响应。
- **记忆重放 (Memory Replay)**: 借鉴RL中的经验回放机制，周期性地从情景记忆中采样旧经验进行“重温”，以巩固学习。

### 3. 记忆更新与整合

- **遗忘曲线模拟**: 引入一种机制来模拟记忆的自然遗忘过程，并根据重要性或使用频率决定记忆的保留与更新。
- **知识蒸馏**: 将情景记忆中的新经验提炼成语义记忆中的新知识或更新现有知识图谱。
- **元记忆 (Meta-Memory)**: 维护一个关于记忆本身的记忆，用于指导记忆的存储、检索和更新策略。*(基于标题与领域常识推断,未读全文)*

## Motivation

Current AI agents often struggle with complex, multi-step tasks due to a lack of effective memory mechanisms. They find it difficult to integrate historical information, learn long-term patterns, or transfer knowledge across different contexts.

Existing models typically rely on limited short-term context windows or simple RAG mechanisms, which restrict their ability to leverage past experiences deeply and adapt to new environments.

## Method

### 1. Memory Encoding and Storage

- **Short-term Memory (Working Memory)**: Utilizes the attention mechanism of Transformers to capture immediate contextual information and encode it into vector representations.
- **Episodic Memory**: Stores sequences of specific agent experiences (observations, actions, rewards), organized using graph structures or time-series databases.
- **Semantic Memory**: Establishes a Knowledge Graph (KG) to store factual knowledge and conceptual relationships, represented through embedding techniques.

### 2. Memory Retrieval Mechanism

- **Context-Aware Retrieval**: Dynamically adjusts retrieval strategies based on the current task and short-term memory content, retrieving relevant information from episodic and semantic memory in parallel.
- **RAG Enhancement**: Combines retrieved memory fragments with current input, feeding them to a large language model via a RAG mechanism to generate more accurate and information-rich responses.
- **Memory Replay**: Adopts the experience replay mechanism from RL to periodically sample old experiences from episodic memory for "revisiting," thereby consolidating learning.

### 3. Memory Update and Integration

- **Forgetting Curve Simulation**: Introduces a mechanism to simulate the natural forgetting process of memory, deciding memory retention and updates based on importance or usage frequency.
- **Knowledge Distillation**: Distills new experiences from episodic memory into new knowledge within semantic memory or updates existing knowledge graphs.
- **Meta-Memory**: Maintains a memory about memory itself, used to guide memory storage, retrieval, and update strategies. *(Inferred from title/venue; full paper not read.)*
