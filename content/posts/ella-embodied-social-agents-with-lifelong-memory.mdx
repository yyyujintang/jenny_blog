## Motivation

当前具身AI智能体在复杂、动态的社交环境中面临挑战，难以维持长期一致的行为和学习能力。
现有方法通常缺乏一个能够支持终身学习和适应性行为的鲁棒记忆系统。
这限制了智能体在需要持续互动和情境理解的社交任务中的表现。

## Contribution

- **终身记忆系统**: 提出一个多模态的终身记忆架构，结合了感知、情景和语义记忆，支持智能体在长期互动中持续学习和适应。
- **具身社交智能体**: 设计了一个名为Ella的具身智能体，能够在虚拟社交环境中进行自然的对话和行为，并利用其记忆系统进行决策。
- **动态环境适应性**: 智能体能够根据记忆中的经验动态调整其行为策略，从而在不断变化的社交场景中展现出更高的参与度和完成率。

## Method

### 1. 感知记忆 (Perceptual Memory)
- **多模态感知**: 智能体通过视觉和听觉传感器收集环境信息，包括其他智能体的行为、对话内容和场景变化。
- **实时编码**: 感知数据被实时编码并存储，作为智能体对当前情境的基础理解。

### 2. 情景记忆 (Episodic Memory)
- **事件存储**: 记录智能体在特定时间、地点发生的具体事件和互动，包括对话、动作和观察。
- **情境关联**: 利用时间戳和空间信息将相关事件关联起来，形成连贯的经验序列。

### 3. 语义记忆 (Semantic Memory)
- **知识图谱构建**: 从情景记忆中提取高层次的抽象知识和规则，构建一个动态的知识图谱 (KG)。
- **概念学习**: 智能体通过归纳和泛化，学习关于世界、其他智能体和自身行为的通用概念。

### 4. 决策与行动 (Decision and Action)
- **记忆检索**: 在决策时，智能体从感知、情景和语义记忆中检索相关信息。
- **行为生成**: 结合检索到的信息和当前目标，通过一个RL或LLM驱动的模块生成适当的对话和具身行为。

## Motivation

Current embodied AI agents face significant challenges in complex, dynamic social environments, struggling to maintain long-term consistent behavior and learning capabilities.
Existing methods often lack a robust memory system that can support lifelong learning and adaptive behaviors.
This limitation restricts agent performance in social tasks requiring continuous interaction and contextual understanding.

## Contribution

- **Lifelong Memory System**: Introduction of a multimodal lifelong memory architecture, integrating perceptual, episodic, and semantic memory to support continuous learning and adaptation in long-term interactions.
- **Embodied Social Agent**: Design of Ella, an embodied agent capable of natural conversation and behavior in virtual social environments, leveraging its memory system for decision-making.
- **Dynamic Environment Adaptability**: The agent can dynamically adjust its behavioral strategies based on remembered experiences, demonstrating higher engagement and completion rates in evolving social scenarios.

## Method

### 1. Perceptual Memory
- **Multimodal Perception**: The agent collects environmental information through visual and auditory sensors, including other agents' behaviors, dialogue content, and scene changes.
- **Real-time Encoding**: Perceptual data is encoded and stored in real-time, forming the agent's foundational understanding of the current situation.

### 2. Episodic Memory
- **Event Storage**: Records specific events and interactions experienced by the agent at particular times and locations, including conversations, actions, and observations.
- **Contextual Association**: Utilizes timestamps and spatial information to link related events, forming coherent sequences of experience.

### 3. Semantic Memory
- **Knowledge Graph Construction**: Extracts high-level abstract knowledge and rules from episodic memory to build a dynamic Knowledge Graph (KG).
- **Concept Learning**: The agent learns general concepts about the world, other agents, and its own behaviors through induction and generalization.

### 4. Decision and Action
- **Memory Retrieval**: During decision-making, the agent retrieves relevant information from perceptual, episodic, and semantic memory.
- **Behavior Generation**: Combining retrieved information with current goals, an RL or LLM-driven module generates appropriate dialogue and embodied actions.
