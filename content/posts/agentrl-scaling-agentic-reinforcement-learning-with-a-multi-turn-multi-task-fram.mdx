## Motivation

Recent advances in large language models (LLMs) have sparked growing interest in building generalist agents that can learn through online interactions.

However, applying reinforcement learning (RL) to train LLM agents in multi-turn, multi-task settings remains challenging. This is primarily due to a lack of scalable infrastructure and stable training algorithms.

## Contribution

- **Scalable Infrastructure**: AgentRL features a fully-asynchronous generation-training pipeline for efficient multi-turn RL.
- **Unified API Interface**: To support heterogeneous environment development in multi-task RL, AgentRL designs a unified function-call based API interface, containerized environment development, and a centralized controller.
- **Cross-Policy Sampling**: The framework proposes cross-policy sampling to encourage model exploration in multi-turn settings.
- **Task Advantage Normalization**: AgentRL introduces task advantage normalization to stabilize multi-task training.
- **Open-Source Implementation**: The AgentRL framework and its algorithms are open-sourced and adopted in building AutoGLM.

## Method

### Infrastructure

1.  **Asynchronous Generation-Training Pipeline**: AgentRL employs a fully-asynchronous pipeline for efficient multi-turn RL training.
2.  **Unified API Interface**: A function-call based API is designed to support heterogeneous environment development.
3.  **Containerized Environments**: Environment development is containerized and managed by a centralized controller.

### Algorithms

1.  **Cross-policy Sampling**: Encourages model exploration in multi-turn interactions, preventing convergence to local optima.
2.  **Task Advantage Normalization**: Stabilizes the multi-task training process, ensuring balanced learning across different tasks.

## Comparison with Previous Work

- **vs. Existing RL Training Methods**: Current methods lack scalable infrastructure and stable algorithms for multi-turn, multi-task LLM agent training. AgentRL addresses these issues with its asynchronous pipeline and unified API.
- **vs. GPT-5, Clause-Sonnet-4, DeepSeek-R1**: AgentRL significantly outperforms these leading closed-source and open-source LLM agents across five agentic tasks.
- **vs. Task-Specific Models**: Multi-task training with AgentRL matches the best results among all task-specific models, demonstrating its versatility and efficiency.
